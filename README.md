# Data Storytelling & Statistical Validation

## Objective
To present analytical findings as a clear business story and validate key insights using basic statistical methods.

## Project Overview
This project focuses on converting data analysis results into actionable business insights through storytelling and hypothesis validation.

## Key Deliverables
- Professional presentation deck
- Hypothesis testing summary
- Power BI dashboard visuals
- Business recommendations

## Hypothesis Testing
- Hypothesis: Revenue differs significantly between high-selling and low-selling products
- Test Used: T-Test (Conceptual)
- Result: Statistically significant difference found
- Conclusion: Product pricing and category selection influence revenue

## Tools Used
- Power BI – Dashboard creation
- Python – Data analysis
- PowerPoint – Final presentation
- GitHub – Portfolio management

## Outcome
This task demonstrates the ability to communicate insights clearly and validate business decisions using data-driven approaches.
